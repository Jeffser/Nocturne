# Contributing Rules

- Before contributing be aware that Nocturne follows the [GNOME Code of Conduct](https://conduct.gnome.org).
- Before contributing code or translations please open an issue explaining what you want to contribute.
- AI generated code is not allowed, if you still plan on using AI make sure it follows the structure of the codebase and that it works.
- Any contributions that makes significant changes to existing files or deletes existing code are subject to additional testing and might get rejected.
